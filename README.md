## Installation 
0. Install dependencies  
``` bash
npm i
```

1. Create collection   
``` bash 
npm run create_collection
```

2. Run API  
``` bash
cd ./frontend
npm run api
```

3. Run Website  
``` bash
# with liveserver run index.html with 'Open with Liveserver'
```
